lets_plot.ggsize¶

lets_plot.ggsize(width, height)¶

Specify overall size of plot.

Parameters
widthint

Width of plot in px.

heightint

Height of plot in px.

Returns
FeatureSpec

Plot size specification.

Examples

1import numpy as np
2from lets_plot import *
3LetsPlot.setup_html()
4np.random.seed(42)
5x = np.arange(50)
6y = np.random.normal(size=50)
7data = {'x': x, 'y': y}
8ggplot(data) + geom_line(aes('x', 'y')) + ggsize(400, 150)
