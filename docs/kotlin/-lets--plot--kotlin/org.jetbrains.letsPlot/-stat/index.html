Stat

object Stat

Stat options to pass as a value of stat parameter of layer functions like:

val n = 100
val rand = java.util.Random(42)
val data = mapOf("x" to List(n) { rand.nextGaussian() })
letsPlot(data) +
    geomArea(stat = Stat.density()) { x="x" }
