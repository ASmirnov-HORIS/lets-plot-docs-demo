Geom

object Geom

Geom options to pass as a value of geom parameter of layer functions like:

val n = 100
val rand = java.util.Random(42)
val data = mapOf("x" to List(n) { rand.nextGaussian() })
letsPlot(data) +
    statDensity(geom = Geom.area()) { x="x" }
