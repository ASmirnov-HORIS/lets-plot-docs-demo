Sampling options.

Used as a value of the sampling parameter in geom_xyz() functions.

val n = 100
val m = 5
val rand = java.util.Random(42)
val data = mapOf("x" to List(n) { rand.nextInt(m) })
letsPlot(data) +
    geomBar(sampling = samplingRandomStratified(3, seed=42)) { x="x" }
